A starter theme called wpBrigade challenge.


Step 1: Upload the theme in Themes Folder
Step 2: Activate the theme. Go to Appearance-> Menus -> Add a menu named 'Secondary Menu'. Then Add two pages 
named membership and account and select 'secondary menu' option for this menu.
Step 3: Then create a Primary menu named, 'Primary Menu', set Primary Menu location and add pages as desired. 
It will be a three tier structure.
Step 4: Then go to 'Themes option' from dashboard, and add slider images as required.
Step 5: Next go to 'Slider Posts' from dashboard, and add posts in it for custom post slider.
Step 6: Next create some posts in wordpress post, and recent posts wil be shown. 
Step 7: Set any image as sticky post to display sticky post.
Step 8: Add tabs content, Top navigation, Disclaimer content from Dashboard.


SIDEBAR INSTRUCTIONS

Step 9: Go to appearnace->widgets->In Sidebar, add widget named, 'Events Widget'. Go to Events custom posts,
add new posst, set date and time and publish the posts.
Step 10: Next craete some categories, assign ur post those categories, and simply add ' Categories List' widget. It will show categories
Step 11: Third search for 'Weather Widget' select city u want.
Step 12: Then add 'Date and time widget'. Select zone and all done!

Footer INSTRUCTIONS

Step 13: For footer Site Map, You must create a footer menu, then search a widget 'Footer Menu Widget' and add 
it in Footer Column 1. 
Step 14: Respectively, add 'Footer Links Widget' in footer column 2.
Step 15: For third column, paste the following code. Links are dealt with shortcodes
